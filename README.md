# Crosshair Way

A premium open-source transparent crosshair overlay for Windows gamers.

Built with Electron. Customizable shape, size, color, opacity, outline, and custom image support. Features dynamic zoom modifier mode triggered by mouse button.

## Features

- **Crosshair Generator** — cross, dot, circle, T-shape with full customization
- **Custom Image Mode** — use any PNG as your crosshair
- **Zoom Modifier** — bind a mouse button to dynamically switch to a smaller crosshair profile
- **Multi-monitor** — place the overlay on any display
- **Hotkey toggle** — global hotkey to show/hide
- **Profiles** — save and load crosshair presets
- **Auto-start** — launch on Windows startup

## Usage

```
npm start       # Launch app
npm run dev     # Launch with --dev flag
npm run build   # Package to dist/ (NSIS + MSI)
```

Windows-only (win32/x64).
